# 📈 投资助手 - Chrome 浏览器扩展

> 财报解读 + 大师选股器 + 内在价值计算器 + 热点资讯 + 股票分析，融合巴菲特/林奇/费雪/芒格/格雷厄姆价值投资策略，AI驱动的全方位投资决策助手。

## ✨ 功能特性

### 🔥 热点资讯
实时追踪市场热点，聚焦关键领域：
- **行业热点**：半导体、新能源、AI科技、机器人、核电、伊朗战况等6大领域
- **公司资讯**：关注公司的最新动态和公告
- **多数据源**：财联社电报、东方财富7×24小时快讯
- **智能过滤**：按领域标签快速筛选，关键词搜索
- **自动刷新**：可配置刷新间隔，实时获取最新资讯

### 🎯 价值投资大师选股器
基于6种价值投资大师策略，AI逐一筛选候选股票：

| 策略 | 核心思想 | 关键指标 |
|------|----------|----------|
| **🏛 格雷厄姆** | 深度价值 · 安全边际 | PE<15, PB<1.5, 股息≥3%, 流动比率≥2 |
| **🏰 巴菲特** | 护城河 · 优质企业 | ROE≥15%, 所有者盈余, 定价权, 管理层 |
| **🔍 彼得·林奇** | PEG · 成长价值 | PEG<1, 公司分类, 盈利增长15-30% |
| **🌱 费雪** | 长期成长 · 15要点 | 研发投入, 利润率, 管理层深度 |
| **⚖️ 芒格** | 理性 · 逆向思维 | ROIC>WACC, 逆向排除, 压力测试 |
| **🌟 综合大师** | 多策略融合 · 严选 | 5大师加权评分，综合排名 |

**选股方式灵活**：
- 📌 **实时搜索提示**：输入股票代码/名称，自动调用东方财富 API 匹配，显示代码、名称、行情、涨跌幅
- 键盘上下选择、回车确认，一键录入
- 已选股票以标签形式展示，可逐个删除
- 支持行业/条件描述：`白酒行业龙头股`、`A股市值超500亿且ROE>15%的股票`
- **流式输出**：实时显示AI分析过程，无需等待

### 🧮 企业内在价值计算器
输入股票名称/代码，自动获取最新财报数据填充参数，支持5种经典估值方法：

| 方法 | 核心公式 | 适用场景 |
|------|----------|----------|
| **💵 DCF折现** | 两阶段FCF折现 + 永续终值 | 成长期/成熟期企业 |
| **🏛 格雷厄姆** | V = EPS × (8.5+2g) × 4.4/Y | 价值型/深度低估 |
| **💳 DDM股息** | V = D1 / (r - g) | 稳定分红的成熟企业 |
| **📐 相对PE/PB** | PE估值 + PB估值 均值 | 有可比同行的企业 |
| **📊 EVA经济附加值** | 价值 = IC + ∑(ROIC-WACC)×IC/(1+WACC)^t | 资本效率型分析 |

**智能数据填充**：
- 📌 输入股票名称 → 自动搜索匹配（东方财富 API）
- 选择后自动拉取：PE/PB/市值/ROE/EPS/BVPS/DPS/FCF/总股本/净债务/WACC
- ✅ 绿色标记 = 自动填充值  🟡 黄色标记 = 建议默认值
- 无法自动获取的参数给出合理建议范围

**结果展示**：
- 内在价值 vs 当前市价 对比
- 安全边际百分比 + 低估/高估/合理判定（🔴高估/🟡合理/🟢低估）
- 可视化柱状条对比
- 计算假设明细

### 📊 结构化财报解读报告
融合四大投资大师分析框架，自动生成10大模块深度解读：

1. **核心业绩概览** - 关键指标同比/环比对比
2. **业务亮点** - 3-5个数据支撑的正面信号
3. **增长驱动力** - 量/价/结构/效率四维分析
4. **不及预期原因** - 深挖根因与持续性判断
5. **投资风险** - 短期+中长期+行业红旗信号
6. **管理层观点** - 分类提炼+诚信检验
7. **投资大师视角** - 护城河/PEG/成长因子/逆向思维
8. **同行业趋势对比** - 横向对比表+趋势分析
9. **行业深度分析** - 周期量化评分+竞争格局
10. **综合评述** - 定调+评级+前瞻信号

### 🔎 股票投资分析
基于投资公司分析框架，全方位深度分析：

**9大分析维度**：
1. 📊 **公司概览与定调** - 核心定位和投资定调
2. 🏭 **行业与商业模式分析** - 行业天花板、竞争格局、护城河评级
3. 💰 **财务稳健性评估** - 关键指标表格 + 综合评价
4. 👔 **管理层质量评价** - 4维度评分（优/良/中/差）
5. 🏢 **大股东/机构持股变化分析** ⭐ - 股东户数、十大流通股东、机构增减持、聪明钱流向
6. 📐 **估值分析** - DCF估值区间、PE/PB/PS对比、安全边际
7. ⚠️ **核心风险提示** - 政策/技术/宏观/关键人风险
8. 🚀 **预期与催化剂** - 市场预期 + 未来6-12个月催化剂
9. 🎯 **投资策略建议** - 明确操作建议 + 目标价位

**自动数据获取**：
- 实时行情数据
- 财务主要指标（利润表、资产负债表、现金流量表）
- 三大现金流关系分析
- 机构持股数据（股东户数、十大流通股东、机构增减持）
- 多年财务指标趋势

## 📦 安装方式

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `earnings-report-extension` 文件夹

## 🚀 使用方式

### 首次使用
1. 点击扩展图标 → 点击 ⚙️ 设置按钮
2. 在「🤖 大模型配置」Tab中添加模型服务
3. 填写 API Key 并保存
4. 切换到「⭐ 关注公司」Tab添加关注股票（可选）

### 热点资讯
1. 切换到「🔥 热点」标签
2. 在「行业热点」子Tab查看实时热点
3. 点击领域标签过滤特定行业
4. 在「公司资讯」子Tab查看关注公司的动态

### 选股器
1. 切换到「🎯 选股器」标签
2. 选择投资策略（格雷厄姆/巴菲特/林奇/费雪/芒格/综合大师）
3. 输入候选股票代码、名称或行业描述
4. 点击「🚀 开始选股分析」
5. 实时查看AI流式分析结果

### 内在价值计算
1. 切换到「🧮 估值」标签
2. 输入股票名称/代码并选择
3. 系统自动填充财务数据
4. 选择估值方法（DCF/格雷厄姆/DDM/相对PE/EVA）
5. 调整参数后点击「🧮 计算内在价值」

### 财报解读
1. 在浏览器中打开财报 PDF 文件
2. 点击扩展图标，自动打开侧边栏
3. 系统自动提取 PDF 内容并生成解读报告
4. 或输入股票代码查询财报并解读

### 股票分析
1. 切换到「🔎 股票分析」标签
2. 输入股票名称/代码并选择
3. 查看自动获取的财务数据和机构持股信息
4. 点击「🚀 开始投资分析」
5. 获取包含9大维度的完整分析报告

## 🏗 项目结构

```
earnings-report-extension/
├── manifest.json              # 扩展配置（Manifest V3）
├── background/
│   └── background.js          # Service Worker，PDF下载+消息路由
├── content/
│   └── content.js             # 内容脚本，PDF检测
├── sidebar/
│   ├── sidepanel.html         # 侧边栏页面（多标签布局）
│   ├── sidepanel.css          # 侧边栏样式
│   ├── sidepanel.js           # 核心逻辑（热点+选股器+估值+财报+股票分析+对话+设置）
│   └── options.html           # 设置页面
├── lib/
│   ├── pdf.min.js             # PDF.js 库
│   └── pdf.worker.min.js      # PDF.js Worker
└── icons/                     # 扩展图标
```

## 🔧 技术栈

- **Chrome Extension Manifest V3** - 最新扩展标准
- **Side Panel API** - 侧边栏展示
- **PDF.js** - PDF 文本提取（本地打包）
- **Web Speech API** - TTS 播报
- **OpenAI Compatible API** - LLM 调用（流式输出）
- **东方财富 API** - 股票搜索 + 实时行情 + 财务数据 + 机构持股
- **财联社 API** - 实时热点资讯
- **纯原生 JS/CSS** - 无额外依赖

## ⚠️ 注意事项

1. **API Key 安全**：存储在 localStorage，不上传到任何服务器
2. **数据隐私**：财报文本和选股请求仅发送到配置的 LLM API
3. **免责声明**：本工具仅供参考，不构成投资建议，投资有风险
4. **机构持股数据**：数据来源于东方财富，可能存在延迟
5. **热点资讯**：资讯内容来自第三方数据源，请自行判断准确性
